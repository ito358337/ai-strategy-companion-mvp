# アプリケーション仕様書 (App Specification)

## 1. アプリ名
AI経営伴走アプリ（MVPライト版）

## 2. アプリの目的
単なる自動診断や採点ツールではなく、AIが伴走者として経営者・経営幹部の頭の中にある「想い」「経験」「強み」「課題」を丁寧な対話を通じて引き出し、整理する。最終的に、経営者が腹落ちし、次の一歩を踏み出せる「売上アップ戦略シート」として言語化・出力（PDF/Markdown）することを目的とする。

## 3. 想定ユーザー
- 自社の戦略を見直したい中小企業経営者、起業家
- モヤモヤとした課題感を抱えている経営幹部・マネージャー

## 4. 利用シーン
- 四半期・半期ごとの経営戦略見直しセッション
- 新規事業立ち上げ前の思考整理
- 経営者自身の壁打ち・内省の時間

## 5. 5章構成
対話は以下の5つの章（フェーズ）で進行する。
1. **戦略編 (Strategy)**：現在地とゴールの確認
2. **戦術編 (Tactics)**：販売プロセスの設計
3. **宣伝編 (Promotion)**：ブランディングとアプローチ
4. **体験編 (Experience)**：顧客体験と感情的価値
5. **自社の経営哲学 (Philosophy)**：自社の経営哲学

## 6. MVPライト版の範囲
- **質問データ**: 各章10問、計50問の厳選されたJSONデータをローカルに配置し、読み込む。
- **対話UI**: 一問一答形式でUIを展開。戻る・次へボタンでの行き来が可能。
- **データ保存**: ユーザーの回答およびAIの要約をデータベース（Supabase等）に保存し、途中再開を可能にする。
- **要約と出力**: 章ごとの簡易要約、全章完了時の総合要約、Markdown/PDF出力機能を実装。
- **履歴管理**: 過去のセッション一覧と詳細の閲覧機能。

## 7. 処理の流れ
1. **質問回答の流れ**: 画面に1問ずつ質問を表示し、テキストエリアで回答を受け付ける。
2. **AI深掘りの流れ**: ユーザーの回答が短い・曖昧な場合、AIが「deepQuestionPrompt.md」を用いて1度だけ深掘り質問を投げかけ、思考を促す。
3. **章ごとの要約**: 1つの章（10問）が完了した時点で、「sessionSummaryPrompt.md」を用いて回答群をプロのビジネス用語を用いて要約し、画面に表示する。
4. **全体要約**: 5章すべてが完了後、全体レビュー画面にて「action90Prompt.md」を用いて90日アクションプランを生成する。
5. **PDF/Markdown出力**: 対話と要約の結果を「pdfSummaryPrompt.md」を用いて成形し、「売上アップ戦略シート」として出力する。
6. **過去セッション保存**: 完了したセッション（途中経過含む）はDBに保存され、一覧からいつでも再開・閲覧可能。

## 8. 将来拡張の方向性
- 各章30問（MVP本格版）、各章100問（Full版）へのシームレスなデータ拡張
- ブラウザ上での質問データベース（QuestionBank）編集管理画面
- 複数ユーザー（マルチテナント）管理および権限設定
- 会社ごとのナレッジ蓄積と、過去データに基づくAIのプロアクティブな提案改善
- リッチなPDFデザインテンプレートの選択機能
